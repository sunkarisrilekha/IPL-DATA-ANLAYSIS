# IPL Data Analysis Dashboard

## Project Overview
Interactive IPL Data Analysis Dashboard developed using Tableau, SQL, and Excel to analyze team performance, player statistics, match outcomes, and tournament insights through dynamic visualizations and KPI reporting.

## Tools Used
- Tableau
- SQL
- Excel

## Dashboard Features
- Team Performance Analysis
- Top Run Scorers and Wicket Takers
- Toss Decision Impact Analysis
- Venue-wise Match Insights
- Win Percentage Comparison
- Interactive Filters and KPI Cards

## Key Insights
- Mumbai Indians recorded the highest overall wins
- Virat Kohli emerged as one of the top run scorers
- Teams batting second showed higher winning percentages in several venues
